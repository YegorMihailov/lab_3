# Алгоритмический мини-пакет

Реализация основных алгоритмов сортировки и структур данных на Python с тестами и бенчмарками.

## Структура проекта

 <pre>
    lab_3/
├── src/
│ ├── benchmark.py # Бенчмарки 
│ ├── constants.py # Константы
│ ├── functions.py # Основные алгоритмы сортировки
│ ├── main.py # Точка входа в приложение
│ ├── queue.py # Реализация очереди
│ ├── stack.py # Реализация стека
│ └── test_cases.py # Генераторы тестовых данных
├── tests/
│ ├── test_functions.py # Тесты алгоритмов сортировки
│ └── test_structure.py # Тесты структур данных
├── requirements.txt
└── README.md
</pre>


## Алгоритмы сортировки

- **Bubble Sort**
- **Quick Sort**
- **Counting Sort**
- **Radix Sort** 
- **Heap Sort**
- **Bucket Sort**

## Математические функции

- **Факториал**
- **Числа Фибоначчи**

## Структуры данных

- **Стек** (LIFO)
- **Очередь** (FIFO)

## Запуск тестов

python src/main.py
